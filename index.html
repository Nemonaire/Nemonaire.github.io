Vision Pro 4DGS Rendering Application — Real-time 4D Gaussian Splatting Visualization

Developed a Swift + Metal rendering application for 4D Gaussian (dynamic 3D Gaussian) scenes on visionOS, extending the open-source MetalSplatter renderer.

  • GPU-accelerated depth sorting and filtering for optimized real-time 4DGS rendering.
  • Gesture-based control for drag, scale, rotate, and hold-to-reset interactions.
  • Spatial audio playback for immersive experience.
MetalSwiftvisionOS4DGS

Houdini 4DGS Plugin — VFX Tool for Editing 4D Gaussian Splatting Models

Developed a Houdini plugin using C++, OpenGL, and Houdini HDK to render and edit 4D Gaussian Splatting models, extending and integrating with the open-source GSOPs framework.

  • SOP nodes to render 4DGS models in both the Houdini viewport and OpenGL export renderer.
  • HDA tools for editing and compositing 4DGS models in VFX workflows.
C++OpenGLHoudini4DGS

4DGS Stylized Rendering — From Realism to Abstraction

Explored the expressive range of 4D Gaussian Splatting, demonstrating how 4DGS can transition from realistic to abstract rendering styles.

  • Wireframe and triangle splat rendering modes for non-photorealistic visualization.
  • Dynamic control of point count and scale.
OpenGLHoudini4DGS

4DGS × Traditional 3D Fusion — Depth-Aware Composition & Cinematic Refocus

Created 4DGS demos integrating 4D Gaussian Splatting with traditional 3D assets, showcasing per-pixel depth compositing and film-style depth-of-field effects.

  • Depth-based blending between 4DGS models and mesh geometry.
  • Refocus control with adjustable focal plane and depth of field.
